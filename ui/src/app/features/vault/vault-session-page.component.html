This tab only

While Unlocked, this tab holds the vault key in memory so items can decrypt. Locked or logout clears it; other browsers or tabs stay locked until you unlock there too.

Lock vault

Ends in-memory access immediately (no need to wait for the idle timer). Ciphertext on the server is unchanged.

Idle auto-lock

Minutes without activity before this tab drops the key. Client-only; not stored on the server. Tap a preset to apply now; while unlocked, a new choice restarts the full idle window from that tap; while locked, your choice applies on the next unlock here. When unlocked, the idle time remaining appears under the green vault status control in the left rail.
