Security

Client-side first

Your vault must be unlocked in this tab. Plaintext is encrypted before upload; the service stores ciphertext and metadata only.

Categories = clarity

Picking a type helps you search and rotate secrets later. It does not weaken encryption, every item gets its own data key inside the vault envelope.
