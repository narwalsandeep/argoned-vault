Why subtypes?

Structured secrets

Each subtype maps to a vault item_type so lists stay scannable and forms match the secret you are storing.

Same crypto everywhere

Argon2id-derived vault key, per-item DEK, AES-256-GCM, unchanged whether you pick Website or Generic. Subtypes only change the form and metadata.

Tip

Eight subtypes are live, Website, Generic, API key, App, Database, SSH, Email, and Software license. More structured types can follow the same pattern.
